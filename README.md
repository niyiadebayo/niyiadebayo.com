# niyiadebayo.com

Static personal site for Adeniyi Adebayo: operator essays on emerging-market cities, marketplaces,
informal systems, distribution, and capital formation, plus a short personal arc.

Pure HTML and one CSS file, no framework, served from GitHub Pages on `main`.

Editorial direction, author voice, the article template, and the publishing process live in one
place: [`AGENTS.md`](AGENTS.md). Read it before editing anything.
